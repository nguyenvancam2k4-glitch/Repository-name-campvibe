# Tóm tắt chức năng frontend CampVibe

CampVibe là website đặt phòng và trải nghiệm cắm trại, glamping nghỉ dưỡng.

## Chức năng người dùng

- Xem trang chủ giới thiệu dịch vụ.
- Tìm kiếm địa điểm glamping.
- Xem danh sách địa điểm.
- Xem chi tiết từng địa điểm.
- Kiểm tra lịch trống và gửi yêu cầu đặt phòng.
- Xem trang xác nhận đặt phòng.
- Xem danh sách trải nghiệm đi kèm như BBQ, lửa trại, picnic, săn mây.
- Xem blog/cẩm nang.
- Xem chi tiết bài viết.
- Gửi yêu cầu tư vấn ở trang liên hệ.
- Xem tài khoản khách hàng.
- Xem địa điểm yêu thích.
- So sánh địa điểm.
- Xử lý trang lỗi 404.

## Chức năng quản trị mô phỏng

- Đăng nhập admin.
- Xem dashboard quản trị.
- Xem thống kê đặt phòng.
- Xem danh sách đơn đặt phòng.
- Xem trạng thái đơn đặt phòng.

## Ghi chú

Hiện tại frontend dùng dữ liệu giả lập. Sau khi làm backend, các service trong `src/services` sẽ được dùng để gọi API thật.
