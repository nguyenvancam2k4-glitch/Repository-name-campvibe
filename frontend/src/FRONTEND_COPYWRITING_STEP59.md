# Step 59 - Customer-facing copywriting

## Mục tiêu

Chỉnh ngôn từ trên giao diện để website giống một sản phẩm thật dành cho khách hàng, không còn cảm giác là trang demo kỹ thuật.

## Đã chỉnh

- Bỏ bớt các từ:
  - PostgreSQL
  - database
  - API
  - demo
  - đồ án tốt nghiệp
  - mô phỏng
- Chuyển sang ngôn từ:
  - đặt lịch
  - tư vấn chuyến đi
  - trải nghiệm glamping
  - hỗ trợ khách hàng
  - thông tin được cập nhật thường xuyên
- Footer mới đã đổi sang ngôn từ thương mại/dịch vụ.

## Test nhanh

Mở các trang public và đọc câu chữ:

```text
/
/places
/places/1
/experiences
/blog
/contact
/compare
/account
/favorites
```

Nếu không còn thấy câu chữ kỹ thuật lộ ra ngoài giao diện khách hàng thì bước này đạt.
